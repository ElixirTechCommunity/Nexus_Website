## Setup

1. `git clone https://github.com/ElixirTechCommunity/Nexus_Website.git`
2. `open project in vscode.`
3. `create your personal branch.`
4. open terminal and run `npm install` in root directory.
5. run `$ npm run build`
6. `open index.html in vsCode.`
7. `start live server.`
